# My Blog - A Dynamic Blog Platform

A modern, dynamic blog platform built with vanilla JavaScript, Handlebars templates, and SCSS. Features a clean, responsive design with dark mode support and interactive tag filtering.

## ✨ Features

- **Dynamic Content Loading** - Articles are loaded and filtered dynamically without page refresh
- **Tag Cloud** - Visual tag representation with size and color based on frequency
- **Author Filtering** - Click on authors to see their articles
- **Dark Mode** - Elegant theme toggle with smooth transitions
- **Fully Responsive** - Works perfectly on all devices
- **Accessible** - Built with accessibility in mind (keyboard navigation, screen readers)
- **Modern CSS** - Uses CSS Grid, Flexbox, and CSS Custom Properties
- **Smooth Animations** - Subtle animations for better user experience
